#  House Price Prediction Using Linear Regression

This project builds a Linear Regression model to predict house prices using square footage, number of bedrooms, and number of bathrooms.

##  Dataset
Kaggle - House Prices: Advanced Regression Techniques  
https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data

##  Features Used
- GrLivArea (Square Footage)
- BedroomAbvGr (Bedrooms)
- FullBath (Bathrooms)

## ⚙️ Technologies
- Python
- Pandas
- Scikit-learn
- NumPy

##  Goal
Predict SalePrice using Linear Regression and evaluate using RMSE and R² score.
